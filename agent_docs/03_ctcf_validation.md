## Что такое CTCF и почему он — ground truth

**CTCF (CCCTC-binding factor)** — цинк-пальцевый транскрипционный фактор,
связывается с ДНК в специфических мотивах (~19 bp). Ключевые факты:

1. **Молекулярный механизм:** Совместно с когезином CTCF формирует петли ДНК
   (loop extrusion model). Петли создают границы между TAD — домены изолированы
   физически за счёт CTCF-заякоренных петель.

2. **Экспериментальное подтверждение:** Деплеция CTCF → слияние соседних TAD.
   Это прямо доказано в нескольких работах (Nora et al. 2017, Cell).

3. **Rao et al. 2014 (наш датасет):** 54% из 12 903 контактных пиков ассоциированы
   с мотивом CTCF. ChIP-seq для GM12878 — эксперимент на тех же клетках.

4. **Следствие для диплома:** Если алгоритм детектирует границы TAD,
   но на них нет обогащения CTCF → границы биологически нерелевантны →
   такой алгоритм не годится как источник разметки для нейросети.

**CTCF — не просто метрика.** Это единственная биологически информированная
валидация, доступная без дополнительных экспериментов на GM12878.
