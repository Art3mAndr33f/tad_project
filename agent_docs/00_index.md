## Быстрый старт для нового участника

1. Прочти `rules.md` (корень проекта) — 5 минут, суть всего проекта
2. Прочти `09_workstreams.md` — узнай свой workstream и его границы
3. Загрузи agent_docs файл своего workstream
4. Работай только в своей зоне ответственности (см. §Ownership)

## Золотые правила параллельной работы

- ❌ Не меняй config/config.yaml без согласования (влияет на всех)
- ❌ Не меняй интерфейс run_<algo>() — только внутреннюю логику
- ❌ Не меняй CONSENSUS_COLORS и ALGO_COLORS
- ✅ Все новые фичи — через новый файл или новую функцию
- ✅ Своя ветка git для каждого workstream
- ✅ rules.md обновляется последним, после merge
